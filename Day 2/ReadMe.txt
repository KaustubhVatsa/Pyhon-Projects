Creating a calculator appliction which takes in an expression from the user and calculates and stores the calculation in a text file
Iterations : 
   Iteration 1 : 
      A basic calculator which takes in 2 numbers and an operator such as + , - , * , \ 
   Iteration 2 : 
      Improve the input function to get multiple numbers from user such as 4 + 5 + 6 
   Iteration 3 : 
      Improve the calculator for more operators
      
Features : 
   a. Show History: Shows the history of all the calculates
   b. Clear Historu : Clears the history 
   c. Bring last result : Brings in the last calculated result
